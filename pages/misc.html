Miniconda and Conda Cheat Sheet

Sample command to create a conda environment with PyTorch:
➥ conda create -n myenv pytorch==1.12.1 torchvision==0.13.1 torchaudio==0.12.1 cudatoolkit=11.6 -c pytorch -c conda-forge

If you get a message similar to the messages below,
>>> 'conda' is not recognized as an internal or external command, operable program or batch file
>>> 'pip' is not recognized as an internal or external command, operable program or batch file
...you may first try to activate conda by running or a similar command to the below:
➥ path-to-miniconda/Miniconda/Scripts/activate
...or you may need to update the PATH variable.

Export an environment using conda:
➥ conda env export > environment.yml

Recreate an environment using conda:
➥ conda env create -f environment.yml

Some remarks:

You cannot pickle lambda functions (unless you have something that uses copy_reg to register how to pickle lambda functions, see this entry

Torch nn parameters are mutable.

'with torch.no_grad():' does not disable gradient computation. Instead gradients are propagated until the first parameter with requires_grad=False. Do not rely on with 'with torch.no_grad():', freeze all the weights manually.

PyTorch - cuda is not deterministic even if you set seed for rng. I will include a short script to my github on using seeds and determinism for PyTorch-cuda.
